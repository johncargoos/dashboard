# CPS Dashboard

A Next.js application with separate admin and carrier dashboards.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app/admin/dashboard` - Admin dashboard routes
- `/app/carrier/dashboard` - Carrier dashboard routes
- `/components/shared` - Shared components used by both admin and carrier

## Routes

- Admin Dashboard: `/admin/dashboard`
- Carrier Dashboard: `/carrier/dashboard`
